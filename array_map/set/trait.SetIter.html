fn is_disjoint<S: SetIter<T>>(&self, other: &S) -> bool where
    T: Hash + Eq, 

Returns true if self and other have no elements in common.

This is equivalent to checking for an empty intersection.

In mathematical notation this is self ∩ other = ∅ or (∀x ∈ self: x ∉ other) ∧ (∀x ∈ other: x ∉ self)
